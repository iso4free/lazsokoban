# lazsokoban
Ещё один клон классической японской логической игры, написанный на FPC/Lazarus.
Возможности:
  * демонстрация прохождения уровня (при наличии файла прохождения)
  * отмена ходов
  * сохранение/загрузка игры (в процессе разработки)
  * таблица рекордов для каждого уровня (в процессе разработки)
  * редактор уровней (частичная функциональность)
  * поддержка скинов (в процессе разработки)
  * поддержка наборов уровней (в процессе разработки)
